Differentiable Extended Kalman Filter

JAX · JIT Compilation · Neural Networks

GitHub Repository

Problem

Classical EKFs are widely used for state estimation, but hand-tuning noise parameters and bias models is time-consuming and can be brittle when sensor characteristics change. The goal was to implement an EKF that is fully differentiable end-to-end so learned components (like bias correction) can be optimized using gradient-based training while preserving the structure and interpretability of model-based filtering.

Technical Approach

Filter Structure

  • State representation: quaternion-based orientation with bias terms
  • Prediction: IMU integration with process noise modeling
  • Update: measurement residual and Jacobian-based correction

Differentiability

  • JAX-based implementation enabling automatic differentiation through predict/update steps
  • JIT compilation for fast batched sequence processing during training
  • Learned bias correction module trained jointly with filter parameters

Numerical Stability

  • Quaternion normalization and careful handling of small-angle approximations
  • Covariance symmetry enforcement to reduce accumulation of numerical error
  • Configurable damping for update steps in high-noise conditions

Results

  • Accuracy: improved tracking stability versus a non-learned baseline under biased IMU conditions
  • Performance: efficient sequence processing using JAX vectorization and JIT compilation
  • Deployment readiness: structure supports embedded-friendly inference due to explicit state updates

Key Challenges

  • Gradient stability: managed exploding gradients by controlling sequence length and update damping
  • Quaternion math: ensured consistent conventions across prediction, update, and evaluation
  • Debugging: validated each component against a classical EKF implementation before enabling learning

Learnings & Future Work

  • Differentiable filtering bridges model-based estimation and data-driven calibration
  • Good numerical practices matter as much as the learning objective
  • Next steps: differentiable square-root filters, better uncertainty calibration, and multi-sensor updates
