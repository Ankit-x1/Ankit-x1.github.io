Physics-Based IMU Intelligence

Raspberry Pi · ONNX · Kalman Filter · Edge AI

GitHub Repository

Problem

Many sensing deployments need high-rate anomaly detection close to the sensor to reduce bandwidth, improve latency, and continue operating with limited connectivity. The goal was to build a system that fuses IMU signals using physics-based filtering while running a compact ML model on low-power edge hardware.

Technical Approach

System Architecture

  • Sensor fusion: Kalman filtering to stabilize estimates and reduce noise
  • ML inference: LSTM-based anomaly scoring compiled to ONNX for deployment
  • Edge runtime: optimized inference loop for continuous streaming at high sampling rates

Data Processing

  • Windowed streaming features with overlap for low-latency updates
  • Per-axis normalization and drift compensation
  • Optional downsampling to match compute budgets while preserving transient events

Deployment Details

  • Model quantization for smaller memory footprint and faster inference
  • Profiling and batching strategies tuned for Raspberry Pi CPU constraints
  • Logging of anomaly scores and filter residuals for debugging in the field

Results

  • Real-time performance: sustained streaming inference at 100-1000 Hz depending on configuration
  • Latency: low millisecond-scale per-window inference on CPU with quantized ONNX runtime
  • Robustness: improved stability by combining learned scores with physics-based residual checks

Key Challenges

  • Compute constraints: balanced model size, window length, and sampling rate to hit real-time limits
  • Sensor noise: tuned filter parameters and validated against vibration-heavy scenarios
  • Edge debugging: added telemetry for filter residuals to isolate failures vs. true anomalies

Learnings & Future Work

  • Combining physics-based filtering with ML improves reliability under domain shifts
  • Quantization and profiling are required to make ML practical on constrained devices
  • Next steps: multi-sensor fusion, online calibration, and better uncertainty estimation for anomaly scores
